EXCEL SPREADSHEET

On the left is an image of an excel spreadsheet project that I worked on. This project required me to set up a spreadsheet that could be used to store the grades of my classmates.

The spreadsheet required me to use functions and equations that could easily average student test scores and show if the student's progress is satisfactory. Since this is a large spreadsheet, the design was important. I had to make it look appealing to the eye while making it easy to navigate and understand.

 

 

MICROSOFT ACCESS DATABASE

While taking INF 202 Intro to Data and Databases, I created this database for my final project. The task was to create relational database implementing the tools that we used throughout the semester.

The most important part of this database is the relationship between the entities. The attributes in each entity had to be linked so a person can easily run a query using SQL. This would provide them an option to view the data attributes that they desire.
